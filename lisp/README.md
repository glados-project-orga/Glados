# LISP Interpreter

## Usage

Start the interpreter from your terminal:

```bash
./glados
```

You can then write your LISP code directly in the interactive shell.
For example:

```lisp
(define two (+ 1 1))
```

The interpreter will parse and evaluate your expression and print the result when appropriate.

---

## Exiting the Interpreter

To quit cleanly, simply type:

```
exit
```
